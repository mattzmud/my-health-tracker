📱 Add to home screen
▼
Save MyWellpath to your phone's home screen for instant access — it launches just like a native app with no browser bar in the way.

🤖

Tap the ⋮ menu in the top right → tap Add to Home screen → tap Add
💻
Desktop (Chrome / Edge)
Look for the install icon in the address bar → click Install
Note: must be opened in Safari on iPhone or Chrome on Android — in-app browsers (from texts or emails) don't support home screen install.
